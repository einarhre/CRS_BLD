# eoh

make_cross-compiler.sh: To build a cross compiler used to build applications that are supposed to run on Windows. Also, a native Windows GCC-compiler has been compiled.
make_pkgs.sh: To cross-compiler packages from source for running on Windows.

build: Directory holding files for the build process.
install: Directory holding native Windows compiler, cross compiler and potentionally other compiled packages.
src: Directory holding sources for the compiler and packages.
